# 30 Movies Page

A small static project that presents a movies page built with HTML, CSS, and JavaScript.

## Overview

This repository contains a single-page static site for a movies-style layout. It uses plain HTML for structure, a stylesheet for layout and visuals, and a small JavaScript file for interactive behavior.

## Features

- Responsive page layout and grid of movie items (implemented in CSS).
- Basic interactivity and behavior implemented in JavaScript.

## Files

- [index.html](index.html) — Main page and entry point.
- [assets/css/style.css](assets/css/style.css) — Project styles and layout rules.
- [assets/js/script.js](assets/js/script.js) — Client-side scripts and interactions.

## Setup & Run

1. Quick preview: open [index.html](index.html) in your browser.

2. Serve locally (recommended for testing fetches or routing):

   - Python 3:

     ```bash
     python -m http.server 8000
     ```

   - Node (http-server):

     ```bash
     npx http-server -c-1 . 8000
     ```

Then open http://localhost:8000 in your browser.

## Development

- Edit styles in [assets/css/style.css](assets/css/style.css).
- Edit interactivity in [assets/js/script.js](assets/js/script.js).
- The page structure lives in [index.html](index.html).

## Deployment

This is a static site and can be deployed to GitHub Pages, Netlify, Vercel, or any static host. For GitHub Pages, push to a repository and enable Pages from the repository settings.
